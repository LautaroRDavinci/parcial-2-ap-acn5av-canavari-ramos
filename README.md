# StyleHub — Sistema de gestión para peluquerías

## Objetivo General (SMART)

Desarrollar en un plazo de 8 semanas un MVP de una aplicación web para peluquerías
que permita a los clientes solicitar turnos online y a los administradores gestionar
servicios, horarios y agenda diaria, con el objetivo de reducir la gestión manual
de turnos y mejorar la organización del negocio.
